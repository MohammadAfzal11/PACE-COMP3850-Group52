👋 WELCOME!

This repository provides automated environment setup for all the 
Privacy-Preserving Record Linkage (PPRL) models.

🚀 SETUP (Choose One):

Option A - Linux/Mac:
  $ chmod +x setup_environment.sh
  $ ./setup_environment.sh
  $ source venv/bin/activate

Option B - Windows:
  > setup_environment.bat
  > venv\Scripts\activate

Option C - Manual:
  $ python3 -m venv venv
  $ source venv/bin/activate  (or venv\Scripts\activate on Windows)
  $ pip install -r requirements.txt

  Solution: Make sure virtual environment is activated!
           source venv/bin/activate

  Problem: Import errors in notebooks
  Solution: See NOTEBOOK_FIXES.md

  Problem: CSV files not found
  Solution: Run scripts from correct directory (cd Afzal/)

  See TROUBLESHOOTING.md for more solutions!
